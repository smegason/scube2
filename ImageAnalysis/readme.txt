Code for " A Scube2-Shh feedback loop links morphogen release and spread to morphogen signaling to enable scale invariant patterning of the ventral neural tube"
Zachary M. Collins1, Anna Cha1, Albert Qin, Kana Ishimatsu, Tony Y.C. Tsai, Ian A. Swinburne, Pulin Li, and Sean G. Megason
bioRxiv 469239; doi: https://doi.org/10.1101/469239

This script is used to extract fluorescence values across the D-V axis of the neural tube from a confocal image stack.
